The dataset that I chose to use was a large dataset released by NASA with information on all detected exoplanets. From this data, I attempted to discern whether or not there was a correlation between planet mass, orbital semi-major axis, number of stars, and eccentricity. Overall, I didn't find any strong correlations, though systems with three stars seemed to have a slightly higher eccentricity than single or double star systems. The data demonstrated more cluster patterns than trends, likely due to bias in detection for planets with small orbital periods, larger masses, and low eccentricities. This led to poor correlations when I tried to apply linear fits.
